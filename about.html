About This Project

Learn more about bigyan.net learning platform

Our Mission

We aim to make learning Nepali accessible and enjoyable for everyone. Our platform combines traditional language learning methods with modern interactive tools to create an effective learning experience.

History

This project is born out of need to have a place for Nepali learners to learn, pronounce and play with the language
